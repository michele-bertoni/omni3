◆ isFinished()

virtual bool Movements::FiniteMovement::isFinished ( const double *  position,
const double *  brakingSpace,
unsigned long  time 
)
pure virtual

This pure virtual method must be overridden by a method that, given the current position, the space of brake and the time, sets _isFinished values to true if the current movement ended for the given axes, false otherwise; if _isFinished is all composed by true values, movements schedule is shifted; for finite movement, it's important to call isFinished method before getSpeed method

Parameters
positioncurrent position of the robot ([m, m, rad])
brakingSpacespace needed by the robot to stop ([m, m, rad])
timecurrent time in milliseconds
Returns
true if all movement components finished, false otherwise

Implemented in Movements::SpeedTimeLinear, Movements::SpaceSpeedLinear, and Movements::SpaceTimeLinear.
